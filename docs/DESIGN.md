# Design notes

## Visual direction

- Dark premium UI
- Clear hierarchy
- Big primary connect action
- Soft borders and cards
- Minimal controls in simple mode

## Simple mode

- Main connect button
- Strategy chips: Auto, Games, Video, Stream
- No technical protocol details exposed

## Advanced mode

- Self-hosted wizard
- Infrastructure server selection
- Security controls
- Split tunneling
- Logs and diagnostics

## Personal cabinet

- Email
- Subscription status
- Expiration date
- Connected devices count
- Invite/share link
