# OwOCloak Windows

Frontend-first Windows VPN client shell built with Flutter.

## What is included

- Simple mode with one main connect action
- Advanced mode with self-hosted and team infrastructure paths
- Personal cabinet section with mock account data
- Clean separation between UI, state, and mock data

## Current scope

This starter is frontend-only.
There is no real VPN backend, tunnel engine, or provisioning logic yet.

## Suggested next step

After unzipping, open the folder in VS Code or Positron and commit the scaffold:

```bash
git add .
git commit -m "Initial Flutter scaffold"
```

Later, when Flutter is installed:

```bash
flutter pub get
flutter run -d windows
```
